BeachBOX
========
BeachBOX is the second iteration of Team 1759P's Push Back robot.
The bot will include 2 pneumatic pistons which will be powered by a solenoids.

Subsystems
==========
* Drive Subsystem
  * 4x 11W motors

Input scheme
============
* Drive train:
  * Arcade drive controls:
    * Axis 3 (left joystick, horizontal axis): turn clockwise (right)/counterclockwise (left)
    * Axis 4 (left joystick, vertical axis): forward/backward
* Outtake:
  * Button.Y = Intake (including arm)
  * Button.A = Top outtake
  * Button.B = Lower outtake
  * Button.X = Bottom outtake (reverse)
